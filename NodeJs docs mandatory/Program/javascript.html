What is javascript?

To put things simply, JavaScript is an object orient programming language designed to make web development easier and more attractive. In most cases, JavaScript is used to create responsive, interactive elements for web pages, enhancing the user experience.

Variables

Javascript has 3 types of variables but only 2 of them is used. Why you may ask? Beacuse Anders says so!
The Three types of variable is:

  • let
  • const
  • var

    • The let statement declares a block-scoped local variable, optionally initializing it to a value. This means the let statment allows you to declare variables that are limited to the scope of a block statement.

    Example

                        var x = 10;
                    // Here x is 10
                    {
                    let x = 2;
                    // Here x is 2
                    }
                    // Here x is 10

    In JavaScript the const means that the identifier can't be reassigned. (Not to be confused with immutable values.
    You can change the properties of a constant object, but you can NOT reassign a constant object:

    Example

                    const car = {type:"Fiat", model:"500", color:"white"};
                car = {type:"Volvo", model:"EX60", color:"red"}; // ERROR

    Callback Function

    Why use Callback Function

    We use javaScript Callback Functions beacuse it can be used synchronously or asynchronously.
    This means we can call a at the same time, or as a relay of information with a time lag.

    What is a callback fucntion

    A JavaScript Callback Function is a function that is passed as a parameter to another JavaScript function, and the callback function is run inside of the function it was passed into.

    In the following example we create a function with a callback function that can be used after a check or somehting else.

    Callback function example

                        function sayHiLater(anyFunctionReference) {
                        anyFunctionReference()
                    }

                    const  myArrowFunction = () => {
                        console.log("Hi there from the anonymous function")
                    }

                    sayHiLater(myArrowFunction);

    In the example above we have a function sayHiLater in witch we give myArrowFunction as a parameter.
    The cool thing is now we can make checks that needs to be true or false before sayHiLater accually runs the myArrowFucntion.
