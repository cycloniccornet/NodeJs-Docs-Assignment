Front-end

HTML

Diffrent tags does diffrent things. This will not be discussed here.
We will look at how class and id can be access. When we make a tag we can add a class or an id to the tag. The class can be given to multiple tags and is mostly used with css.

adding a class to a div

                        div class="codedisplay" /div

We can now call the div using a '.' to tell its a class + name of class. we will look at this in the css part. When we make a tag we can also add a id to the tag. The id is uniqe and can only be added to one tag.

adding a id to a div

                        div id="codedisplay"/div

We can now call the div using a '#' to tell its a id + name of class. we will look at this in the css part.

CSS

In the html segment we talked about how id and class's work. Now we will look at how to access both in css.
Using the syntax from html we can access the class and the id as follows.

Accessing class through css.

                        .codedisplay {
                        #Do some styling to a class.
                        }


Accessing id through css.

                        #codedisplay {
                        #Do some styling to a class.
                        }

Jquery

jQuery is a JavaScript library designed to simplify HTML DOM tree traversal and manipulation, as well as event handling, CSS animation, and Ajax.

Basic jquery syntax

                        $(selector).action()

                        A $ sign to define/access jQuery
                        A (selector) to "query (or find)" HTML elements
                        A jQuery action() to be performed on the element(s)

As mentioned above we can use the selector to access class'es and id's.
